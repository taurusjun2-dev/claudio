# 音乐品味：二次元

## 核心偏好：ACG 音乐

热爱动画、游戏、Vocaloid 相关的日本音乐，注重旋律性和情感爆发力。

### 动画配乐 & OST
- **梶浦由记** — Kalafina、FictionJunction、魔法少女小圆、空之境界。恢弘的管弦乐 + 女声合唱，史诗感和宿命感
- **泽野弘之** — 进击的巨人、罪恶王冠、Aldnoah Zero。燃向电子摇滚，德语歌名是标志
- **横山克** — 四月是你的谎言、機動戦士高达。钢琴旋律极佳
- **牛尾宪辅** — 声之形、莉兹与青鸟。极简电子氛围，细腻到骨头里
- **菅野洋子** — 星际牛仔、攻壳机动队。什么风格都能玩的天才

### J-Pop / Anisong 歌手
- **Aimer** — 沙哑声线，Fate 系列主题曲常客
- **LiSA** — 燃曲女王，刀剑神域、鬼灭之刃
- **YOASOBI** — 小说音乐化，夜に駆ける、アイドル
- **ヨルシカ** — 文学感歌词 + 清爽吉他，夏天专属
- **ずっと真夜中でいいのに** — 独特声线和复杂编曲
- **Official髭男dism** — 旋律天才，每一个副歌都上头
- **Eve** — 独特的世界观，呪術廻戦 OP
- **米津玄师** — Lemon、KICK BACK，每一首都是现象级
- **RADWIMPS** — 你的名字、天气之子，野田洋次郎的词绝了

### Vocaloid / 虚拟歌姬
- **wowaka** — 已故天才，Rolling Girl、World's End Dancehall
- **DECO*27** — 初音未来御用，吸血鬼、幽灵法则
- **Mitchie M** — V家调教巅峰
- **ツミキ** — 快节奏炸裂风
- **cosMo@暴走P** — 超高 BPM，初音未来的消失

### 游戏 OST
- **尼尔：机械纪元** — 废墟都市的悲伤，人声编曲无可替代
- **女神异闻录 5** — Acid Jazz，时髦值拉满
- **塞尔达传说** — 旷野之息的钢琴静谧，王国之泪的恢弘
- **原神** — HOYO-MiX 陈致逸，璃月的民族管弦乐尤其出色
- **东方 Project** — 同人音乐宝库，凋叶棕、岸田教团

### 同人音乐 & 独立
- **Mili** — 多语言 + 古典训练，Goblin Slayer OP
- **やなぎなぎ** — 透明感，来自风平浪静的明天
- **KOKIA** — 天籁之音，游戏主题曲专业户
- **Revo / Sound Horizon** — 物语音乐，每一张专辑都是一部歌剧

## 不喜欢
- 流水线量产的异世界厕纸动画 OP（电吉他速弹 + 高音吼叫模板）
- Vocaloid 中过度尖锐的调教
- 中文翻唱的日文歌（原版更好听）
- 纯说唱
- 过于吵闹而无旋律的金属核

## 情绪对应
- 需要燃 → LiSA、泽野弘之、cosMo@暴走P
- 需要感动 → Aimer、横山克、尼尔 OST
- 需要放松 → ヨルシカ、菅野洋子、塞尔达
- 需要沉浸 → 梶浦由记、牛尾宪辅、Mili
