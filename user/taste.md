# 音乐品味

## 核心圈层
二次元 / 动漫音乐爱好者

## 喜欢的风格

### 动漫主题曲（OP/ED）
- 热血系：紅蓮華（LiSA）、残酷な天使のテーゼ、シュガーソング
- 治愈系：Aimer、やなぎなぎ、坂本真綾
- 近年新番：YOASOBI《アイドル》、ado《うた》、Mrs. GREEN APPLE

### Vocaloid / 虚拟歌手
- 初音ミク：ryo、wowaka、じん、Kenshi Yonezu（米津玄師早期）
- 经典 P 主：ダンテの「炉心融解」、supercell、DECO*27
- 现代风：Kikuo、Orangestar、n-buna（ヨルシカ前身）

### 同人 / 东方 Project
- 上海アリス幻樂団原曲
- FELT、UI-70、GET IN THE RING
- 东方爵士编曲：A-ONE、秋霜玉系列

### 游戏 OST
- KEY 系列：Clannad、Little Busters!、Angel Beats!（麻枝准）
- NieR Automata（岡部啓一）
- Persona 系列（目黒将司）
- 原神、崩坏星穹铁道

### 创作型动画系 Artist
- YOASOBI：双声道编曲，小说改编，高辨识度
- ヨルシカ（Yorushika）：n-buna 作曲，文学气质
- 米津玄師：从 Vocaloid P 主转型，Lemon / Pale Blue
- Aimer：空灵沙哑，Fate 系列常客
- LiSA：鬼滅 / SAO 主题曲，舞台能量强

## 偏好特征
- 喜欢人声有辨识度、旋律记忆点强
- 接受电子 + 管弦混编
- 喜欢有故事感的歌词（即使听不懂日语也感受得到情绪）
- 偏好 BPM 在 120-160 的流畅节奏，深夜听偏慢
- 不抗拒中文动漫 OST（五月天×动漫联动、二次元国风）

## 不喜欢
- 无脑重复的机械电子舞曲
- 过于 idol 味（AKB48 风格流水线）
- 人声高度处理到失真失去情感

## 近期不要推荐
- 《紅蓮華》（听太多了）
- 《アイドル》（循环过多，暂时回避）

## 场景偏好
- 早晨通勤：YOASOBI / 米津玄師 轻快曲
- 工作专注：游戏 OST 纯器乐（NieR / Persona）
- 深夜放松：Aimer / ヨルシカ / KEY 系
- 情绪激动：LiSA / 热血 OP
